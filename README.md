# 介绍
---
这是我的个人博客，里面记录了一些学习笔记、经验和技巧。具体被细分到了每个目录.

均采用 gitbook 编写的文档，查看博客内容，需要有如下步骤：

1. 前置条件，需要有node环境并且安装了gitbook，具体安装细节请查看nodeplugin下的文档

1. 复制根目录的 node_modules 到子目录。

2. 在子目录中执行 gitbook serve
