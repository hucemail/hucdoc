匿名类型

  1. 匿名类型通过var关键字定义
  2. 它通过初始值来判断实际的类型，一经初始化类型就被确定，后面将无法改变它的类型
  3. 无法通过反射来获取信息
  4. 它无法用作形参类型或字段属性的类型，只能在方法内部作为变量的类型使用

如下代码是错误的：

    public void Test()
    {
        var i = 10;
        Console.WriteLine(i.GetType());
        i = "s";
    }
