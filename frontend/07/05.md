# 前言

这小节介绍ng-的一些指令，有如下：


* ng-app:模块
* ng-controller：控制器
* ng-cloak：解决闪屏
* ng-model:绑定元素值，用于表单
* ng-bind:绑定元素值
* ng-value：绑定元素值，可写表达式
* ng-show：为true显示，false则隐藏
* ng-hide：为false显示，true则隐藏
* ng-if：为true执行，false则不执行
* ng-true-value：为true时使用的值
* ng-false-value：为false时使用的值
* ng-options:用于操作下拉列表
* ng-click:点击事件
* ng-repeat:遍历
* ng-src：
* ng-href:
* ng-class：
* ng-style：
