1. CMD规范

淘宝的玉伯大牛搞了个seajs出来，并声称这个规范是遵循CMD规范的，因此CMD规范就此引出。

CMD规范定义：一个文件就是一个模块

define(function(require, exports) {
  exports.each = function (arr) {
    // 实现代码
  };

  exports.log = function (str) {
    // 实现代码
  };
});

通过 exports 就可以向外提供接口。

define(function(require, exports) {
  var util = require('./util.js');

  exports.init = function() {
    // 实现代码
  };
});
