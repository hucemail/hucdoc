# 对象的种类

对象大致分为三种：

1. JS内置对象，如：Number、Date、Array、RegEx等等
2. 宿主对象,如：window、document等等
3. 用户自定义对象

# 对象的创建

对象的创建方式有三种：

1. 字面量创建方式
2. 构造函数创建方式
3. Object.create创建方式

对象的访问方式有两种：

1. 点操作符访问
2. 中括号访问，当对象属性名是不符合规范或者是动态的就必须只能使用中括号访问
3. 当访问一个不存在的属性返回undefined
4. 级联访问
